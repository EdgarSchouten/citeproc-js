CSL Bibliography Samples

When viewed in Firefox with Javascript enabled, this page will display a set of example bibliographies that illustrate some of the capabilities of the citeproc-js citation formatter. For more details, see the CSL website and the processor manual.

A few comments on the bibliography listings below. These are generated from a single set of database entries; the formatting you see is entirely controlled by style code fed to the CSL processor inside your browser. In this page, the style code has been handled using the DOM (Document Object Model, a standard mechanism for parsing structured documents, available in all modern Web browsers). For server-side use, the processor can also use the E4X parsing engine, which is available natively in the Rhino, Spidermonkey and Tracemonkey Javascript implementations.

If parsing fails and you do not see bibliography listings below, please send mail to biercenator@gmail.com, so we can adjust things to work with your browser.
