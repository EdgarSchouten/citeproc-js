CSL Bibliography Samples

When viewed in Firefox, Safari or Google Chrome with Javascript enabled, this page will display a set of example bibliographies that illustrate some of the capabilities of the citeproc-js citation formatter, which implements CSL 1.0. For more details, see the CSL website and the processor manual.

A few comments on the bibliography listings below. These are generated from a single set of database entries; the formatting you see is entirely controlled by style code fed to the CSL processor inside your browser. When viewed in Internet Explorer, Safari or Google Chrome, the CSL style code is read using the DOM (Document Object Model, a standard mechanism for parsing structured documents). For server-side use and in the Firefox browser, the processor uses the E4X parsing engine instead, which is available natively in the Rhino, Spidermonkey and Tracemonkey javascript implementations.

If parsing fails and you do not see bibliography listings below, please send mail to biercenator@gmail.com, so we can adjust things to work with your browser.
